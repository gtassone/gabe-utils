A sampling of java util code written by me.

packages
--------

attribute: A simple XML-based rules engine. Processes objects against attribute constraints to conditionally trigger actions.

bit: something with bitmasks? 

chain: managed chain of responsibility based on MonitorDispatchThread

concurrent: concurrency/ thread management utils including MonitorDispatchThread, a managed collection->task execution thread.

data: data import/export utils including extensible database importers for multiple file types and data sources.

gui: 'fluent' builder pattern for Swing GridBag layout

xml: SAX parser base class simplifies XML parsing in java
